## Como manter este changelog

Sempre que houver mudanças relevantes, registre em uma das categorias abaixo:

### Categorias sugeridas

- **Adicionado**
  - para novas funcionalidades, arquivos ou módulos

- **Alterado**
  - para mudanças em funcionalidades existentes

- **Corrigido**
  - para correções de bugs

- **Removido**
  - para funcionalidades ou arquivos removidos

- **Segurança**
  - para correções ou melhorias relacionadas à segurança
